jQuery googleslides

A jQuery plugin to display any shared Google Photos Album.

Usage:

$('#demo1').googleslides({
  albumid: 'gYCTmRk5ft6aCYdm6'
});

Usage:

$('#demo2').googleslides({
  albumid: 'V4fE9oCsydPaEPu47',
  random: true,
  time: 2000
});

Usage:

$('#demo3').googleslides({
  albumid: 'AFugdhSBmUB7djWNA',
  imgmax: 100,
  random: true,
  time: 2000,
  fadespeed: 500
});

Setup

  1. Add a reference to jquery.googleslides.css and jquery.googleslides.js: 
    <link href="src/jquery.googleslides.css" rel="stylesheet" type="text/css" />
<script src="src/jquery.googleslides.js" type="text/javascript"></script>
  2. Add a container div <div id='slides'></div> element to your body.
  3. Start the slides: 
    $('#demo1').googleslides({
  albumid: 'gYCTmRk5ft6aCYdm6'
});

Finding your albumid

  • To find your albumid, open an album from the Albums tab, on the Google Photos page.
  • When the album comes up, click Share on the top navbar and then Create link on the popup.
  • You will see albumid in the Url, like so

Finding albumid from any other shared album

  • To find albumid, open album using the shared link.
  • When the album comes up, click More Options on the top navbar and then Options on the dropdown.
  • You will see albumid in the Url, like so

Options

Property Default Description
albumid gYCTmRk5ft6aCYdm6 The id of shared Google Photos Album.
imgmax 320 Max size, in pixels, of the photos.
maxresults 100 Max number of photos to show from the album.
random false Randomize the order the photos are displayed in.
albumlink true Link the photos to the Google Photos Album page.
time 5000 Time to show each photo before advancing (in milliseconds).
fadespeed 1000 Fade in/out speed (in milliseconds).
